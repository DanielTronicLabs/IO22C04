# IO22C04
This arduino library is for IO22C04 board

## Microcontroller
ATmega32u4 (pro micro)
ATmega328 (pro mini)
## INPUTS

### Switches
- SW1_PIN
- SW2_PIN
- SW3_PIN
- SW4_PIN

### Terminals
- IN_1_PIN
- IN_2_PIN
- IN_3_PIN
- IN_4_PIN

## OUTPUTS

### Relays
- RELAY1_PIN
- RELAY2_PIN
- RELAY3_PIN
- RELAY4_PIN
